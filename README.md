
# 🛒 공구리: 대학생 공동구매 플랫폼

**함께 사고 나누는 생활 편의 서비스**

> **공구리**는 대학교 기숙사 및 자취생들을 위한 **공동구매 커뮤니티 플랫폼**입니다.
> 대량 구매의 부담은 나누고, 실시간 채팅으로 편하게 소통하세요!

---

## 📌 서비스 소개

대학생들은 경제적인 이유로 생필품을 저렴하게 구매하길 원하지만, 대용량 제품을 혼자 사기엔 부담이 큽니다.
공구리는 **학교 인증 기반**의 커뮤니티 플랫폼으로, **공동구매, 채팅** 기능을 통해 알뜰한 대학생활을 돕습니다.

---

## ✅ 주요 기능

### 🛍 공동구매 기능

* 제목, 내용, 가격, 모집 인원 등 입력 후 게시물 작성
* **채팅방 자동 생성**으로 실시간 소통 가능
* **참여자 관리**: 입금 확인, 강퇴 등

### 💬 실시간 채팅

* 게시물 기반 자동 채팅방 생성
* 공동구매 참여자 간 원활한 실시간 소통


### 👤 프로필 기능

* 사용자의 과거 게시물 확인을 통해 **신뢰도 판단**

### 🔐 회원가입 / 로그인

* **대학교 웹메일 인증 기반 가입**
* 학생만 이용 가능한 안전한 커뮤니티 제공
